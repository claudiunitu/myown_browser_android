# MyOwn Browser Android

A basic, WebView-based Android browser with ad-blocking and other privacy features.

## Features

*   **Ad Blocking:** Blocks ads using a host-based approach.
*   **Third-party Cookie Control:** Option to block third-party cookies.
*   **Permission Management:** Control website access to camera, microphone, and location.
*   **File Downloads:** Supports downloading files from the web.
*   **Customizable Homepage:** Set your own home page.
*   **Basic Navigation:** Go back, forward, and refresh.
*   **URL Bar:** Manually enter URLs or search queries.
*   **Search:** Search the web using DuckDuckGo.

## Getting Started

### Prerequisites

*   Android Studio
*   Android SDK

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/myown_browser_android.git
    ```
2.  Open the project in Android Studio.
3.  Build the project.

## Usage

1.  Run the app on an Android device or emulator.
2.  Enter a URL or search query in the URL bar and tap the "Go" button.
3.  Use the settings button to configure ad-blocking, cookie settings, and permissions.
